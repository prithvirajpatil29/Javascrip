Static Method..!

JS provides static methods that belongs to the class instead of an onstance of an that class. These methods are called directly on the class itself.

Points to remember..!

  • The sttaic keyword is used to declare a static method.
  • The sttaic method can be any name.
  • A class can contain more than one static method.
  • If we declare more than one static method with a similar name, the JS always invokes the last one.
  • We can use this keyword to call a static method within another static method.
  • We can;t use this keyword directly to call a static method within the non-static method
  • In such case, we cam call the static method either using the class name or as the property of the constructor
