Javascript Encapsulation

It is process of binding the data with functions acting on that data. It allows us to control the data and validate it.

Points to remember..!
  • User let and const keyword to make data members private.
  • Use setter method to set the data and getter method to get that data

b Encapsulation allows us to handle an object using the following properties

Read/write - Here, we use setter methods to write the data and getter methods to read data

Read - In this case, we use getter methods only

write - In this case, we use setter methods only
